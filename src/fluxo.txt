O protocolo HTTP-
é um protocolo que se chama: Hypertext transfer protocol
ele define como as requiseções são estruturadas
o http segue um modelo cliente servidor
o cliente envia uma requisição http, o servidor processa e envia uma resposta http
como é uma requisição http? -
1- Método
2- URL
3- headers
4- Body(opcional)


Exemplo: 
o verbo (metódo) é um POST ou GET PUT DELETE PATCH 
A URL pode ser https://api.sp.com
Chamamos de URL ou POST
O tipo de header que eu posso receber:
O contenty-type:aplication/json 
pode ser outros tipos: application/text, form-data
Podemos determinar o que vamos enviar e o que vamos receber

O body pode ser: json- objeto serializado em texto
o body pode ser: um texto convencional
o body pode ser: um form data
o body pode ser: .excel, .csv, .zip, .exe, .png, jpg., .mp4
